Tech Tools

Developer Tools:

Developer Tools

(or "development tools" or short "DevTools") are programs that allow a developer to create, test and debug software.

Current browsers provide integrated developer tools, which allow to inspect a website. They let users inspect and debug the page's HTML, CSS, and JavaScript, allow to inspect the network traffic it causes, make it possible to measure its performance, and much more.

Learning and Education

    Online Courses and Tutorials

  • Udemy: Courses on various programming languages and technologies
  • Coursera: University-style courses on computer science and software development.
  • edX: Offers online courses from institutions like MIT and Harvard.

    Documentation and Reference

  • W3Schools: Tutorials and references on web development languages
  • JavaScript.info: In-depth guide and tutorials on JavaScript.
  • MDN Web Docs: Comprehensive resource for web development, including HTML, CSS, and JavaScript

Code Editors and IDEs

  • Visual Studio Code: Highly customizable, supports many languages and extensions.
  • Sublime Text: Lightweight and fast code editor.
  • Atom: Open-source code editor with a focus on modularity.
  • IntelliJ IDEA: Powerful IDE, particularly strong for Java, Kotlin, and other JVM languages.

Version Control Systems

  • Git: Distributed version control system, essential for modern development.
  • GitHub: Hosting service for Git repositories with collaboration features.
  • GitLab: Integrated CI/CD platform with Git repository management.
  • Bitbucket: Git repository hosting with built-in CI/CD.

Testing Tools

Database Management Tools

  • Jest: JavaScript testing framework with a focus on simplicity.
  • Mocha: Flexible testing framework for JavaScript.
  • JUnit: Popular testing framework for Java.
  • Selenium: Tool for automating web browsers for testing purposes.
  • pgAdmin: Management tool for PostgreSQL databases.
  • MongoDB Compass: GUI for managing MongoDB databases.
  • MySQL Workbench: Visual tool for MySQL database design, development, and administration.
